# Nodeseek Max-iSen

用于增强 NodeSeek/DeepFlood 论坛体验的 Tampermonkey 用户脚本。

## 安装

在已安装 Tampermonkey 的浏览器中打开以下地址，然后确认安装或更新：

<https://raw.githubusercontent.com/EISEN0516/nodeseek-pro-userscript/main/Nodeseek%20Pro.user.js>

也可以从 [Greasy Fork](https://greasyfork.org/zh-CN/scripts/588182) 安装。

## 更新方式

脚本元数据已配置 GitHub Raw 更新地址。Tampermonkey 检查更新时会读取 `main` 分支中的脚本版本号；提交新版本后即可从 GitHub 更新。

本项目不保存论坛账号、密码、Telegram Bot Token 或其他通知服务密钥。此类配置只保存在用户自己的 Tampermonkey 存储中。

## 开发约定

每次修改脚本后递增 `@version`，运行语法检查，再提交到 `main` 分支。公开发布到 Greasy Fork 前应单独确认发布内容。
